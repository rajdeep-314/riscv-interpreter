# Instructions


## Integer Register-Immediate Instructions

Formats:

```
1. INST rd, rs1, imm
2. INST rd, imm
```

Currently implemented:

- Register-Immediate instructions
    - addi
    - xori
    - ori
    - andi
    - slli (shift left logical immediate)
    - srli
    - srai (shift right arithmetic immediate)
    - slti
    - sltiu
    - lui

- Register-Register instructions
    - add
    - ...
